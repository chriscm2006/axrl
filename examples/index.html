AXRL Examples

Example: A basic web page test

This example shows the test for example.com, where the test produced 2 results. One of the results is "failed", and the second is "passed".

{
  "@context": "https://axrl.org/context/web-v1.json",
  "@type": "WebPageTest",
  "testSubject": "http://example.com",
  "results": [
    {
      "@type": "TestFailure",
      "impact": "Critical",
      "description": "Web pages must have a title",
      "remediation": "Add a title to the web page",
      "helpUrl": "https://dequeuniversity.com/rules/axe/3.1/document-title",
      "node": {
        "@type": "DOMNode",
        "codeSnippet": "<html>...</html>",
        "selector": "html"
      }
    },
    {
      "@type": "TestSuccess",
      "helpUrl": "https://dequeuniversity.com/rules/axe/3.1/color-contrast",
      "node": {
        "@type": "DOMNode",
        "codeSnippet": "<h1>Welcome</h1>",
        "selector": "h1"
      }
    }
  ]
}

Example: Multiple test for a web page

A single web page being tested multiple times. This can be used when using DOM Observer to track changes to the page. Scope is used to denote the element(s) that were changed and re-tested

{
  "@context": "https://axrl.org/context/web-v1.json",
  "@type": "HTMLPage",
  "tests": [
    {
      "@type": "WebPageTest",
      "time": "2019-01-31T13:36:18+0000",
      "scope": "*",
      "results": [
        {
          "@type": "TestFailure",
          "impact": "Critical",
          "description": "Web pages must have a title",
          "remediation": "Add a title to the web page",
          "helpUrl": "https://dequeuniversity.com/rules/axe/3.1/document-title",
          "node": {
            "@type": "DOMNode",
            "codeSnippet": "<html>...</html>",
            "selector": "html"
          }
        },
        {
          "@type": "TestSuccess",
          "helpUrl": "https://dequeuniversity.com/rules/axe/3.1/color-contrast",
          "node": {
            "@type": "DOMNode",
            "codeSnippet": "<h1>Welcome</h1>",
            "selector": "h1"
          }
        }
      ]
    },
    {
      "@type": "WebPageTest",
      "time": "2018-01-31T13:36:18+0000",
      "scope": ".new-element",
      "results": [
        {
          "@type": "TestSuccess",
          "helpUrl": "https://dequeuniversity.com/rules/axe/3.1/color-contrast",
          "node": {
            "@type": "DOMNode",
            "codeSnippet": "<p class='new-element'>Some new content</p>",
            "selector": ".new-element"
          }
        }
      ]
    }
  ]
}
